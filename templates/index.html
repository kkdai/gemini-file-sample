如何取得 API Key？

  1. 造訪 Google AI Studio
  2. 登入您的 Google 帳號
  3. 點擊 "Create API Key" 建立新的 API Key
  4. 複製 API Key 並貼到上方的輸入框
  5. 點擊「儲存 API Key」
